

Overview
BytePlus Image Editing API allows you to modify, enhance, or transform images using AI-powered editing capabilities driven by text prompts and parameters.

Authentication
To utilize this API, you must subscribe to the service and obtain an Ocp-Apim-Subscription-Key. This key should be included in the request headers to authenticate your API calls.

Endpoint
Base URL:
https://gateway.pixazo.ai/byteplus/v1/getEditImage

Request Details
Headers
Content-Type: application/json
Cache-Control: no-cache
Ocp-Apim-Subscription-Key: YOUR_SUBSCRIPTION_KEY
Models
seededit-3-0-i2i-250628
seedream-4-5-251128
Request Body Parameters
Parameter	Type	Required	Default	Description
model	string	No	seededit-3-0-i2i-250628	The ID of the model to call. You can activate a model service and query the model ID. An endpoint ID can also be used to call a model.
prompt	string	Yes	-	Text description (prompt) used to edit images. Describes the desired changes to be made to the input image.
image	string	Yes	-	The image to be edited. Accepts either Base64 encoding or an accessible URL.
Image URL: Ensure the URL is accessible.
Base64: Must be in the format data:image/<format>;base64,<content> (format in lowercase, e.g., data:image/png;base64,...).
response_format	string	No	url	Format of the returned image.
Options: "url" (downloadable JPEG link), "b64_json" (Base64-encoded JSON string).
size	string	No(Yes for seedream 4.5)	adaptive	Dimensions of the generated image. Currently only adaptive is supported. The system compares the input image size with predefined sizes and selects the closest match, prioritizing minimal aspect ratio differences. For Model seedream-4-5-251128, Size supported 2K, 4k. Total pixel range: seedream 4.5：[2560x1440=3686400, 4096x4096=16777216]
seed	integer	No	-1	Random seed to control generation randomness. Range: [-1, 2147483647]. If -1 or not set, a random seed is generated automatically. Use the same seed for consistent results.
guidance_scale	float	No	5.5	Controls how much the text prompt vs. input image influences the output. Range: [1, 10]. Higher values = stronger text prompt influence, weaker input image influence.
watermark	boolean	No	true	Whether to add a watermark.
false: No watermark.
true: Adds "AI-generated" watermark in the bottom-right corner.

Example Request
Here is an example using curl:

curl -v -X POST "https://gateway.pixazo.ai/byteplus/v1/getEditImage" -H "Content-Type: application/json" -H "Cache-Control: no-cache" --data-raw "{
    \"prompt\": \"Make the cat eye blue\",
    \"image\": \"https://pub-582b7213209642b9b995c96c95a30381.r2.dev/byteplus/1757499948018-hntkjsg9kj.jpg\",
    \"guidance_scale\": 6,
    \"seed\": 42
}"

Error Handling
If any errors occur during the API call, the response will include an error message detailing the issue. Common errors may include invalid subscription keys or missing request parameters. Ensure all required fields are correctly filled and formatted.

Security Considerations
Keep your API subscription key secure. Do not expose it in client-side code or public repositories.
Rotate your subscription key periodically to maintain security.
