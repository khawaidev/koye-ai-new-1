

## THIS  if the flow for the llm to trigger feature for 2d games:




1.AI chats with user to know more details about the game.


2.AI asks more questions about the game design.


3.AI asks about the game's initial character design.

4.AI asks more about the game's main./player character design.

5.AI generates a prompt for text to image for the character design accordingly to the above chat with the user and user descriptions.(prompt should be much more detailed , should be less than 1000 letters but more than 900 letters)

6.AI shows the generated prompt to the user and asks user for the confirmation of the prompt, and if yes proceed to 7 and then if no then go back to 4.


7.AI shows the user the credit cost for the image gen.if user confirms it then proceed to step 8, and if no then go back to 4.

8.AI triggers(here only single one image, front facing(default) or use users description if users described the view of the characters) the image gen and shows the loading animation for the image gen (the beaming animations) to the user, and if done then proceed to 9 and if any error than proceed to 8.5

8.5:here the ai shows the nature of the error , and asks the user to try again or option to contact the developers, [gives the option the link] (which when clicked a pop up from the bottom comes with a input field for the user to enter their problem and a btn to submit), and if the user choose to try again then go back to 6.
